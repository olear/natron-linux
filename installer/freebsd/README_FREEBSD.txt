Natron and Plugins requires the following:

- glew
- openimageio
- opencolorio
- expat
- qt4
- boost-libs
- ffmpeg
- pixman

You can install the required packages using 'pkg'.

pkg install glew openimageio opencolorio expat qt4 boost-libs ffmpeg pixman

Important! Run installer as root, sudo ./Natron_installer.
