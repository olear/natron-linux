Natron Linux64 Build
====================

Supported/Tested distros:

 - CentOS/RHEL 6.x/7.x (yum install libGLU)
 - Fedora 20 (yum install libGLU)
 - Ubuntu 12.04/14.04
 - Debian 7.5 (apt-get install libegl1-mesa)
 - openSUSE 13.1
 - Mageia 4.1
 - Slackware 14.1

This build should work on any distro released after 2010.

Usage:

Extract TGZ and run 'Natron'.

Installation:

Run 'Install.sh' as root.

Uninstall:

Run /opt/Natron-<VERSION>/Uninstall.sh as root.

Please report any problems with this build.
