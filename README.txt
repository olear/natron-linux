Natron for GNU/Linux x86_64
===========================

See the 'docs' folder for documentation.

Requirements
============

The following distros are known to work:

 - CentOS/RHEL 6/7 
  - yum install libGLU
 - Fedora 20 
  - yum install libGLU
 - Ubuntu 12.04/14.04
 - Debian 7.5
 - openSUSE 13.1
 - Mageia 4.1
 - Slackware 14.1

Usage
=====

Extract TGZ and run 'Natron' from folder.

Install
=======

Run 'Install.sh' as root from folder.

Uninstall
=========

Run /opt/Natron-0.9/Uninstall.sh as root.
