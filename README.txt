Natron for GNU/Linux x86_64
===========================

See the 'docs' folder for documentation.

Requirements
============

The following distros are known to work:

 - CentOS/RHEL 6/7 
  - yum install libGLU
 - Fedora 18/19/20 
  - yum install libGLU
 - Ubuntu 12.04/14.04
 - Debian 7.5
 - openSUSE 13.1
 - Mageia 4.1
 - Slackware 14.1

Install
=======

Make installer executable and run it.

chmod +x installer_file_name
./installer_file_name
