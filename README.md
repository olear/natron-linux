Natron on Linux64
=================

Various build scripts for Natron on Linux64.

Build
=====

Setup enviroment:
sh build-prep.sh

Build core libraries:
sh build-core.sh

Build Qt libraries:
sh build-qt.sh qt4

Build Natron/openfx-io/openfx-misc:
sh build-app.sh qt4

Build Natron from workshop and openfx-io/openfx-misc from master:
sh build-app-workshop.sh
