Natron on Linux
===============

Binaries for Natron on Linux.

Requirements
============

The following distros are known to work:

 - CentOS/RHEL 6/7
 - Fedora 18/19/20
 - Ubuntu 10.04/12.04/14.04
 - Debian 7.5
 - openSUSE 13.1
 - Mageia 4.1
 - Slackware 14.1

Notes
=====

On some distros Natron requires additional software to function.

**CentOS/RHEL/Fedora:**

```
yum install libGLU
```

**Ubuntu 10.04:**

```
apt-get install libxcb-shm0
```
